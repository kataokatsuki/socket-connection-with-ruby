# Test Socket Connection with Ruby

## What you can do?
You can test socket connection with ruby.
- TCP
- TLS over TCP

### TCP 
Run below in order from the top.

```
ruby tcp/server.rb
ruby tcp/client.rb
```

### TLS 
Run below in order from the top.

```
ruby tls/create_key.rb
ruby tls/server.rb
ruby tls/client.rb
```
